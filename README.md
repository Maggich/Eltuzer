# El Tuzer - Студия мебели и салона

Современный веб-сайт мебельного магазина с админ-панелью для управления контентом.

## Технологический стек

- **Frontend**: React 18 + TypeScript
- **Backend**: FastAPI (Python)
- **База данных**: PostgreSQL
- **Контейнеризация**: Docker & Docker Compose

## Особенности

- ✅ Полностью адаптивный дизайн для всех устройств
- ✅ Админ-панель для управления товарами и категориями
- ✅ Аутентификация и авторизация
- ✅ Загрузка изображений для товаров
- ✅ Современный дизайн в стиле неона (голубой на черном фоне)
- ✅ Полная контейнеризация проекта
- ✅ **Landing страница для таргетинга** с формой и отправкой в WhatsApp

## Быстрый старт

### Требования

- Docker и Docker Compose установлены на вашей системе

### Установка и запуск

1. Клонируйте репозиторий или используйте текущую директорию

2. Запустите проект через Docker Compose:

```bash
docker-compose up --build
```

3. После запуска проект будет доступен по адресам:
   - Frontend: http://localhost:3000
   - **Landing страница (для рекламы)**: http://localhost:3000/landing или http://localhost:3000/offer
   - Backend API: http://localhost:8001
   - API документация: http://localhost:8001/docs

## Создание первого администратора

Для создания первого администратора выполните одну из следующих команд:

### Способ 1: Через скрипт (рекомендуется)

```bash
docker-compose exec backend python create_admin.py
```

Скрипт попросит ввести данные администратора (можно использовать значения по умолчанию).

### Способ 2: Через API

```bash
curl -X POST "http://localhost:8001/api/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "email": "admin@eltuzer.com",
    "password": "admin123"
  }'
```

⚠️ **Важно**: После создания администратора обязательно измените пароль на более безопасный!

## Использование панели управления

1. Перейдите на http://localhost:3000/login
2. Войдите с учетными данными:
   - **Логин**: `admin`
   - **Пароль**: `admin123`
3. В панели управления вы можете:
   - Добавлять, редактировать и удалять товары
   - Добавлять, редактировать и удалять категории
   - Загружать изображения для товаров

⚠️ **Важно**: После первого входа обязательно измените пароль на более безопасный!

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация нового пользователя
- `POST /api/auth/login` - Вход в систему
- `GET /api/auth/me` - Получить текущего пользователя

### Товары
- `GET /api/products` - Список товаров
- `GET /api/products/{id}` - Получить товар по ID
- `POST /api/products` - Создать товар (требует авторизации)
- `PUT /api/products/{id}` - Обновить товар (требует авторизации)
- `DELETE /api/products/{id}` - Удалить товар (требует авторизации)
- `POST /api/products/{id}/upload` - Загрузить изображение (требует авторизации)

### Категории
- `GET /api/categories` - Список категорий
- `GET /api/categories/{id}` - Получить категорию по ID
- `POST /api/categories` - Создать категорию (требует авторизации)
- `PUT /api/categories/{id}` - Обновить категорию (требует авторизации)
- `DELETE /api/categories/{id}` - Удалить категорию (требует авторизации)

## Landing страница для таргетинга

Проект включает специальную landing страницу, оптимизированную для рекламных кампаний в социальных сетях (Instagram Stories, Facebook и т.д.).

### Доступ к странице

- **URL**: http://localhost:3000/landing или http://localhost:3000/offer
- Страница открывается **без Header** для максимальной конверсии

### Функционал

- ✅ **Форма обратной связи** (имя и телефон)
- ✅ **Автоматическая отправка в WhatsApp** после заполнения формы
- ✅ **Таймер акции** для создания срочности
- ✅ **Социальные доказательства** (количество клиентов, опыт работы)
- ✅ **Адаптивный дизайн** для мобильных устройств
- ✅ **Маркетинговые элементы** (CTA, бейджи, анимации)

### Настройка номера WhatsApp

1. Откройте файл `frontend/src/pages/Landing.tsx`
2. Найдите константу `WHATSAPP_NUMBER` в начале файла
3. Замените значение на ваш номер (только цифры, без + и пробелов)
   ```typescript
   const WHATSAPP_NUMBER = '79991234567'; // Ваш номер
   ```

### Использование для рекламы

1. **Instagram/Facebook Stories**: Используйте ссылку `/landing` в рекламных кампаниях
2. **Swipe Up**: При настройке Stories добавьте ссылку на landing страницу
3. **Таргетированная реклама**: Используйте `/offer` как альтернативный URL

### Дополнительные возможности

Вы можете настроить:
- Текст заголовка и описания
- Время таймера акции
- Социальные доказательства (цифры)
- Цветовую схему в `Landing.css`

## Остановка проекта

Для остановки всех контейнеров:

```bash
docker-compose down
```

Для остановки и удаления всех данных (включая базу данных):

```bash
docker-compose down -v
```

## Разработка

### Backend разработка

Для разработки backend без Docker:

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend разработка

Для разработки frontend без Docker:

```bash
cd frontend
npm install
npm start
```

## Безопасность

⚠️ **Важно**: Перед развертыванием в продакшене:

1. Измените `SECRET_KEY` в `docker-compose.yml`
2. Используйте сильные пароли для администраторов
3. Настройте HTTPS
4. Ограничьте доступ к админ-панели
5. Настройте резервное копирование базы данных
