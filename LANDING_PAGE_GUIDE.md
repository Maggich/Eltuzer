# Инструкция по использованию Landing страницы

## Как открыть страницу

1. Убедитесь, что проект запущен:
   ```bash
   docker-compose ps
   ```
   Все контейнеры должны быть в статусе "Up"

2. Откройте в браузере один из URL:
   - http://localhost:3000/landing
   - http://localhost:3000/offer

3. Если страница не открывается:
   - Проверьте консоль браузера (F12 → Console)
   - Убедитесь, что frontend контейнер работает: `docker-compose logs frontend`
   - Попробуйте перезапустить frontend: `docker-compose restart frontend`

## Настройка номера WhatsApp

1. Откройте файл `frontend/src/pages/Landing.tsx`
2. Найдите строку 7:
   ```typescript
   const WHATSAPP_NUMBER = '79991234567'; // ЗАМЕНИТЕ НА ВАШ НОМЕР!
   ```
3. Замените `79991234567` на ваш номер (только цифры, без + и пробелов)
   - Пример: для номера +7 (999) 123-45-67 используйте `79991234567`

## Как работает форма

1. Пользователь заполняет форму (имя и телефон)
2. При нажатии "Отправить в WhatsApp":
   - Форма валидируется
   - Открывается WhatsApp с готовым сообщением
   - Показывается сообщение об успешной отправке
   - Появляется кнопка перехода на основной сайт

## Решение проблем

### Страница не открывается
- Проверьте, что frontend контейнер запущен
- Проверьте логи: `docker-compose logs frontend`
- Попробуйте пересобрать: `docker-compose up --build frontend`

### Форма не отправляется
- Проверьте консоль браузера на ошибки (F12)
- Убедитесь, что номер WhatsApp настроен правильно
- Проверьте, что браузер не блокирует всплывающие окна

### WhatsApp не открывается
- Убедитесь, что номер WhatsApp настроен правильно
- Проверьте, что у вас установлен WhatsApp
- Попробуйте открыть ссылку вручную: `https://wa.me/79991234567`

## Использование для рекламы

1. Используйте URL `/landing` или `/offer` в рекламных кампаниях
2. Для Instagram Stories: добавьте ссылку в Swipe Up
3. Для Facebook: используйте в таргетированной рекламе
4. Страница оптимизирована для мобильных устройств
